# Semantic3D кратко

Набор наружных лазерных сканов с покомпонентной семантической разметкой для сегментации точек. Основные классы: man-made terrain, natural terrain, high/low vegetation, buildings, hardscape, scanning artifacts, cars/trucks.

Особенности
- Станционные LiDAR‑сканы → высокая плотность рядом со станцией, разрежение и окклюзии вдали.
- Классы распределены неравномерно, доминируют покрытия/земля — нужно взвешивание или выборки.
- Объём данных большой: важны даунсэмплинг/батчинг, нормализация координат и интенсивности.
- Артефакты и шум усложняют разметку; границы объектов часто неточны.

Минимальный препроцесс: проверить файл, нормализовать RGB и интенсивность, центрировать/масштабировать координаты, почистить артефакты, даунсэмплировать и разбить на train/val/test с нужными аугментациями.
