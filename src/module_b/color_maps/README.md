# Цветовые карты и визуализация

Пример окраски точек по скалярному значению и сравнение matplotlib (статичная картинка) с plotly (интерактив). Ссылки на интерактивные html лежат в `outputs/`.

Кратко
- Диапазон RGB: либо 0–255 (int), либо 0–1 (float); нормализация обязательна перед colormap.
- Цвет по величине: нормализовать значение → подать в colormap (viridis и т.п.) → получить RGB.
- Matplotlib подходит для отчетных картинок, Plotly — для интерактивного осмотра (вращение/подписи).
- Цвет можно задавать руками: списком rgb(R,G,B) или по координате/расстоянию после нормализации.
