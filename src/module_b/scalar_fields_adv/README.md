# Скалярные поля: продолжение

Продолжение задач по скалярным полям: константа, умножение/сложение, сглаживание (гаусс и скользящее), преобразование в RGB, статистика, нормализация, интерполяция, фильтрация по диапазону, использование поля как координаты. Скриншоты — в `outputs/`.

Что запомнить
- Нормализация в [0,1] облегчает визуализацию и последующую обработку.
- Большой σ в гауссовом фильтре размывает детали; скользящее среднее даёт более грубое усреднение.
- Пропуски стоит сначала интерполировать (иначе визуализация и координатная замена ломаются).
- Фильтрация по маске: строим булеву маску по диапазону и применяем её к точкам/полю.
- Convert to RGB переводит значения поля в цвета для наглядности.
