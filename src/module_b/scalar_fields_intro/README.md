# Скалярные поля: базовые операции

Набор упражнений: константное поле, умножение/сложение, гауссово сглаживание, фильтрация по диапазону, анализ статистики, замена координаты значением поля. Скрины лежат в текущей папке.

Локальный конспект
- Add constant / Multiply / Add number позволяют быстро сдвигать или масштабировать поле.
- Gaussian filter даёт мягкое сглаживание; Smooth делает простое усреднение.
- Filter by value оставляет точки в выбранном диапазоне скалярного поля.
- Compute stat. parameters выводит основные метрики (min/max/mean/std).
- Set SF as coordinate подменяет одну из XYZ значениями поля для быстрой визуализации рельефа.
- Gradient/Interpolate помогают заполнить пропуски и оценить резкие переходы.
