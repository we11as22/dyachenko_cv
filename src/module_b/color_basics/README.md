# Базовые цветовые операции в CloudCompare

Короткие пробы: однотонная окраска и градиент по высоте. Скриншоты в этом каталоге.

Полезные заметки
- Цвета в облаке хранятся как RGB; поддерживаются PLY, LAS/LAZ, PCD, E57 и др.
- Height Ramp красит по активному скалярному полю (обычно Z), а Set Color задаёт единый оттенок.
- Без активного поля Ramp не сработает; можно выбрать нужное поле в Properties/Scalar fields.
- Цветовую схему можно инвертировать или перевести в серый, если нужно подчёркивать контраст.
