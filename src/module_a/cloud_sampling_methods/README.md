# Сравнение выборок: random, voxel, FPS

В одном облаке проверены три стратегии уменьшения точек. Визуализации и ссылки на облака лежат в `outputs/`.

Итоги
- FPS лучше всего сохраняет края и общую форму при малом числе точек, но работает медленно (минуты на большой N).
- Voxel grid уравнивает плотность и чистит шум, детализация зависит от размера вокселя.
- Random субсэмплинг почти мгновенный, но может потерять тонкие элементы.

По времени на тестовом наборе: random ≈ 0.003s, voxel ≈ 0.16s, FPS ≈ 114s.
