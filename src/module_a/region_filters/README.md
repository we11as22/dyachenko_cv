# Фильтрация облака точек

Тест трёх простых фильтров: bbox, порог по высоте и радиус вокруг точки. Визуалки и ссылки на облака лежат в `outputs/`.

Количество точек (пример): исходное 158 953 → bbox 3 838 → height 16 227 → radius 3 175.

Замечания
- BBox быстро обрезает сцену, но режет всё, что лежит на границе прямоугольника.
- Порог по Z отделяет «землю» и «надземное», но ошибается на наклонных участках.
- Радиусное выделение удобно для локальных объектов; чувствительно к выбору центра/радиуса.

Лучший результат даёт комбинированный пайп: bbox → radius/height.
