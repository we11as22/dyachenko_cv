# Субсэмплинг облаков точек

Короткое сравнение трёх способов уменьшения облака: случайный выбор, пространственная сетка и октодерево. Скриншоты до/после лежат в `pictures/`.

Основные наблюдения
- Spatial даёт самую аккуратную форму при урезании числа точек примерно втрое.
- Octree неплохо держит крупные детали, но сглаживает мелочи на грубых уровнях.
- Random самый быстрый, но сильнее рвёт края при большом уменьшении.

Числа (пример): исходное облако 158 953 точек → Random 58 953, Spatial 65 984, Octree 76 770.
