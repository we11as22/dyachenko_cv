# Ручная сегментация в CloudCompare

Три примера выделения областей в одном облаке (до/после скрины в `pictures/`). Один сегмент выбран случайно, второй — без земли, третий — перекрашенные сферы.

Наблюдения
- Когда объекты легко различимы глазом, ручной Segment даёт чище маску, чем авто‑алгоритмы.
- Метод удобен для исправления ошибок автоматической сегментации и точной подрезки границ.
- Минусы: медленно, зависит от оператора, разные люди проведут границу по‑разному.

CloudCompare сохраняет атрибуты точек при выделении в новое облако; удалённые точки уносят свои атрибуты из исходного.
