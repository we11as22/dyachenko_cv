# PointNet для классификации 3D-объектов

Реализация классификатора облаков точек на основе архитектуры PointNet для датасета ModelNet10.

## Структура

- `train_pointnet.py` — основной скрипт обучения модели с валидацией
- `pointnet_model.py` — определение архитектуры сети (блоки выравнивания и классификатор)
- `pointnet_data.py` — загрузка и предобработка данных из .off файлов
- `pointnet_visuals.py` — инструменты для визуализации результатов и предсказаний
- `conf.yaml` — конфигурационный файл с гиперпараметрами
- `smoke.py` — быстрая проверка работоспособности пайплайна

## Использование

### Обучение модели

```sh
uv run train_pointnet.py
```

Модель обучается на train-сплите, валидируется на test-сплите. Лучший чекпоинт сохраняется в `artifacts/pointnet.pt`, метрики и графики — в `artifacts/`.

### Визуализация результатов

```sh
uv run pointnet_visuals.py --ckpt artifacts/pointnet.pt
```

Скрипт загружает сохранённую модель, делает предсказания на тестовых примерах и сохраняет визуализации в `artifacts/`.
