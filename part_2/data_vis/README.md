# Визуализации датасета

Небольшая выборка PNG-превью облаков точек из `../data/`.

На **одной** картинке слева и справа показано **одно и то же облако** из одного PLY-файла:
- слева — раскраска по `scalar_Label`;
- справа — только геометрия.

Разные PNG-файлы соответствуют **разным** PLY. Они могут выглядеть похоже, потому что в датасете много однотипных клапанов в близкой ориентации. Скрипт специально выбирает более разные облака по числу точек/форме и меняет ракурс для каждого файла.

Сгенерировать заново:

```sh
cd part_2
source ~/envs/.venv/bin/activate
python scripts/visualize_point_clouds.py --max-files 6
```

Явно указать файлы:

```sh
python scripts/visualize_point_clouds.py --files valve_0001_lidar_classes.ply valve_0123_lidar_classes.ply valve_0330_lidar_classes.ply
```

Полный датасет `data/` в git не коммитится.
