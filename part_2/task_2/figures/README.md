# Визуализации для сдачи

В этой папке лежит небольшая выборка PNG-визуализаций из полного прогона `task_2`.

Полная папка `visualizations/` содержит 4440 изображений и не коммитится в git. Для отчёта и защиты оставлены по два примера для каждого типа геометрии:

- `tube_*` — трубчатые объекты;
- `sphere_*` — сферические/объёмные формы;
- `complex_*` — сложная геометрия;
- `plane_*` — плоские сегменты.

Каждая картинка показывает исходный сегмент облака точек и реконструированную mesh-поверхность.
